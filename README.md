# Description

**This project follows the [Unity Core Creative roadmap](https://learn.unity.com/pathway/creative-core?version=6.3) on Unity Learn.**

# Technologies used
* Unity Editor v6000.2.11f1
* Vim 9.1 with dependencies:
  * OmniSharp LSP server
  * Unity tool [kotpodlawkoy sln generator](https://github.com/kotpodlawkoy/kotpodlawkoy.sln.csproj.generator)

# Work done

In the context of this project, the following work was done:

* With textures and materials (including emissive materials)
* With shaders (including vertex shaders: flag animation)
* With lighting (including light probes and reflection probes)
* With animations (including working with keyframes and importing ready-made ones)
* Visual effects (and their development from scratch using VFX Graph and Particle System)
* Camera settings and post-processing
* Audio settings and mixing + settings for various effects
* User interface (UI)

The project can be compiled in Unity
